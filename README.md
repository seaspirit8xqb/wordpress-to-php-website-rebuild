# WordPress / Hosted Builder to Custom PHP Website Rebuild

A case study for rebuilding a business website from a hosted builder/WordPress-style workflow into a custom HTML/PHP website with editable content, SEO pages, product sections, clean URLs, and a safer deployment workflow.

## Project Overview

The original website was managed through a hosted website-builder / WordPress-style workflow. The goal was to rebuild it into a more maintainable custom website structure that was easier to edit, easier to optimize for SEO, and easier to deploy safely.

The rebuild focused on turning the website into a custom HTML/PHP structure with editable content areas, reusable page sections, SEO-ready pages, product/service sections, responsive layouts, and a cleaner production workflow.

## What I Built

- Rebuilt the public website using custom HTML/PHP
- Converted hosted-builder style content into maintainable page sections
- Created responsive desktop and mobile layouts
- Built reusable page sections and templates
- Added editable content structure for easier updates
- Added blog/SEO content structure
- Added product/service listing sections
- Added contact and location sections
- Added Google Maps integration
- Added clean URL routing
- Added sitemap and robots setup
- Prepared the website for Google Search Console
- Created a safer clone/staging-to-production update workflow
