# Photomory Website Rebuild & CMS Migration

A case study for rebuilding and migrating a public business website from a hosted builder/WordPress-style workflow into a custom PHP/HTML website with a lightweight admin CMS.

## Overview

This project focused on making the Photomory public website easier to maintain, more flexible for content updates, and better prepared for SEO and production use.

The work included editable page content, blog/SEO article management, product data for a mini shop, visual content sections, clean URLs, sitemap setup, Google Search Console preparation, and a clone-to-production deployment workflow.

## Key Features

- Custom PHP/HTML website structure
- Lightweight admin CMS
- Editable page content
- Blog and SEO article management
- Topic-based editorial blog section
- Mini shop with product categories and cart actions
- Google Maps and contact page integration
- Clean URL routing
- Sitemap and robots setup
- Google Search Console preparation
- Mobile responsive layout

## Role

Web Developer / CMS Builder

## Tech Stack

PHP, HTML, CSS, JavaScript, JSON data files, responsive design, SEO basics
